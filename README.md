# Sales Dashboard Analysis (Power BI)

## Project Overview
This project analyzes 1000+ sales transaction records to generate KPI-based business insights using Power BI.

## Tools Used
- Power BI
- Microsoft Excel
- DAX (Data Analysis Expressions)

## Key Insights
- Region-wise revenue comparison
- Monthly sales trend analysis
- Identification of top-performing products
- KPI performance tracking

## Dashboard Features
- Interactive Filters & Slicers
- Revenue KPI Cards
- Product Performance Visualization
- Monthly Trend Analysis

## Business Impact
This dashboard helps stakeholders monitor revenue performance, identify growth opportunities, and make data-driven business decisions.
